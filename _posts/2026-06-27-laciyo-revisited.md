# Week 14: Philosophy

## The Vulgate's Legacy

I wrote about Latin Vulgate in June 2025. The Latin of the people. Not Cicero's polished prose, but the language that actual humans spoke.

## The Evolution

Latin → Vulgate Latin → Portuguese/Spanish/French/Italian/Romanian

One language, many children. Each child thinks it's unique. Each child carries the parent's DNA.

## The Code Switch

```
// Modern JavaScript
const love = "amor";
console.log(love); // amor

-- Latin Vulgate
dilectio est amor
-- love is love

-- Portuguese
o amor é amor
-- same truth, different tongue
```

## The Personal Legend

The Vulgate was translated so people could read scripture in their own language. This blog is... what? So I can think in public? So I can prove I showed up?

Jerome translated the Bible. I translate thoughts into markdown.

The scale differs. The impulse doesn't.

## The Thread

Latin → Portuguese → This post in English

The Vulgate won. We're all reading the translation now.
