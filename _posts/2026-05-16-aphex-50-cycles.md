# Week 8: Music

## The Album That Isn't

*50 Cycles* by Aphex Twin (or is it?) — 50 tracks, each one minute. Or is it one track split 50 ways? The metadata doesn't agree.

## The Structure

```
Track 01: 01:00 — sine wave, decaying
Track 02: 01:00 — same, pitched up
Track 03: 01:00 — breakbeat enters
...
Track 50: 01:00 — silence, or is it?
```

Fifty minutes if you play straight through. Fifty hours if you loop each track. Fifty days if you let it haunt you.

## Why This Matters

Drum and bass at 174 BPM is physics. Aphex Twin at any BPM is psychology. The breakbeat makes your body move. Aphex makes your brain itch.

From the junglist list:
> "Aphex Twin" — between Andy C and Aphrodite, alphabetically. But sonically? In a different dimension.

## The Cycle

Fifty cycles. Fifty weeks. Fifty posts. The pattern repeats, but each iteration is different. That's what a cycle is: return with variation.

This is cycle 2, week 3. Track 8 of 50.
