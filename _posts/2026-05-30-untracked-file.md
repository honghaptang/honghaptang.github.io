# Week 10: Wildcard

## Git Status

```
$ git status
On branch main
Untracked files:
  setup.sh
```

There's a file I haven't committed. `setup.sh`. It's been there for weeks. Maybe months.

## What's In It

Probably initialization code. Environment setup. Something that runs once and never again. That's the thing about setup: you do it, then you forget it.

## The Metaphor

Untracked files are the thoughts I haven't written. The posts I started and abandoned. The ideas that exist in the working directory of my head but never make it to commit.

## The Choice

I could:
1. Add it — make it part of the history
2. Ignore it — add to `.gitignore`, pretend it doesn't exist
3. Delete it — it served its purpose, let it go

I'm choosing option 4: leave it there. A reminder that not everything needs to be tracked.
